函数声明定义

声明函数用如下的书写格式:

declare 函数名(参数1, 参数2, ...)

声明了的函数才可以被调用。声明只是宣告存在一个函数，但不知道其内容如何，所以还需要定义函数。

定义函数用如下的书写格式:

function 函数名(参数1, 参数2, ...)
    private 局部变量1, 局部变量2, ...
    ......
    return 返回值

    ......
 end 返回值

返回值是可选的，若没有返回值，系统自动返回0。局部变量只在函数内部起作用，可以不声明。函数调用可以递归。

　

　

若定义一个函数时，编译器发现该函数没有声明，就会按定义的参数自动声明一个函数。

　

下面是一个求最大公约数函数的例子：

function Gcd(a, b)
  if a > b    //总将小值赋给a
    t = a

    a = b

    b = t
  endif
  if (b % a) : a = Gcd(b % a, a)
end a

可以定义不限参数个数的函数，方法是在function后加open关键字。此时函数名后的参数表无意义，需要自己用Pop语句将参数从栈中弹出。

如定义一个求最大值的函数，使用了不限参数个数的方法：

function open Max()
  pop n        //n是编译器压入栈中的参数个数
   if n<=0: return
  pop a
  for i=2 to n
    pop b
    if a<b: a=b
  next
end a

　

调用时形如：a=Max(b,c,d)
