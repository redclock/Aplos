Aplos Script 简介

什么是Aplos?

    Aplos脚本语言是一种容易学习的强大语言。它有丰富的控制结构，动态变量类型，可自定义函数，数组动态创建和自动回收。编译后的脚本可在控制台和图形窗口模式运行。

如何使用：

    Aplos的源文件写在扩展名为.apl的文本格式文件中（但不强制使用.apl扩展名）。但源文件不能运行。需要用脚本编译器将源文件编译为二进制字节码的.spt文件。然后用运行器将二进制脚本运行起来。

使用APLOS进行脚本开发，有两种选择：    

  1. 使用提供的脚本编辑器，编辑，编译，运行脚本，十分方便。

    参见教程《使用脚本编辑器开发APLOS脚本》

    　

  2. 也可以不使用编辑器，用任何文本编辑器（比如记事本）编写源文件并保存。再用命令行的编译器aplos.exe进行编译。
