语言简介

Aplos语言源文件以行来分隔。每一行的内容可能为：

  • 空行或注释行

  • 编译指令

  • 以关键字开头的命令或控制语句

  • 赋值语句

  • 表达式，一般为函数调用，也可以是1+1一类的表达式单独成行

    一个源文件中可以用include语句来将其他文件包含进来。一个脚本程序由零个或多个函数再加上一个主执行体构成。请参看函数声明和定义。

函数之外的语句都是主执行体。

脚本将在主执行体的第一个语句开始执行

一个最初级的脚本的例子（只一行）：

    Writeln（"Hello World!"）

用也可以有许多函数:

  function f1(a)

   end

   function f2(b)

  end

   //main
   private x
   f1(x)
   f2(10)
