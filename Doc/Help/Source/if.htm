If 语句结构

if的语法形如:

if 条件表达式1

  语句

  语句

  ...

[else if 条件表达式2

  语句

  语句

  ...]

[else if 条件表达式3

  语句

  语句

  ...]

...

[else

  语句

  语句

  ...]

endif

带方括号的部分表示是可选的,如果 "条件表达式1" 成立,就接着下边的语句,否则转入"else if 条件表达式2" 部分,依此类推,直到遇到endif为止

例子:

if a>0

  writeln "正数"

else if a<0

  writeln "负数"

else

  writeln "0"

endif

 

简写形式:

if语句下的语句块若只有一条语句,则可以使用单行简写形式:

if .... : 语句

如:  if n<0:exit

注意if语句的单行形式不能有else部分
